# Scoring Rubrics Reference

Consolidated scoring criteria for all CET Skill modes.

---

## Writing Scoring Rubric (Official)

四六级作文采用**总体印象评分制**，满分 15 分，分 5 个固定档次。阅卷先确定各档次标准样卷，对照样卷给分。四级与六级同档次样卷有级别难度差异。

### 评分档次

| 档次 | 分值 | 评分标准 |
|------|------|---------|
| 14 分档 | 13–15 分 | 切题，表意清晰，行文通顺连贯，基本无语言错误，仅零星小失误 |
| 11 分档 | 10–12 分 | 切题，表意清晰，语句连贯，存在少量语言错误 |
| 8 分档 | 7–9 分 | 基本切题，部分内容表意模糊，行文勉强连贯，语言错误数量多，含多处严重错误 |
| 5 分档 | 4–6 分 | 基本切题，表意混乱，连贯性差，大量严重语言错误 |
| 2 分档 | 1–3 分 | 逻辑散乱、思路紊乱，句式破碎，绝大多数句子存在严重错误 |

### CET 分数换算

作文满分 106.5 分（710 分的 15%）。近似换算：

| 原始档 | 原始分 | 换算分 ≈ |
|--------|--------|----------|
| 14 分档 | 13–15 | 90–106 |
| 11 分档 | 10–12 | 70–85 |
| 8 分档 | 7–9 | 50–63 |
| 5 分档 | 4–6 | 28–42 |
| 2 分档 | 1–3 | 7–21 |

### 诊断维度（辅助分析，非官方评分）

用于在反馈中拆解问题来源，帮助用户理解得分偏低的具体原因：

| 诊断维度 | 说明 |
|---------|------|
| 内容切题度 | 是否回应题目要求 |
| 结构完整度 | 开头-主体-结尾是否清晰 |
| 逻辑连贯性 | 段落间、句子间衔接 |
| 语法准确性 | 语法错误的密度和严重程度 |
| 词汇准确性 | 用词是否恰当、搭配是否正确 |
| 句式多样性 | 简单句/复合句/复杂句的搭配 |
| 语言自然度 | 是否自然的英文表达（非模板堆砌） |

## Translation Scoring Rubric (Official)

四六级翻译同样采用**总体印象评分制**，满分 15 分，分 5 档。同档次四级与六级译文评分标准存在级别难度差异。

### 评分档次

| 档次 | 分值 | 评分标准 |
|------|------|---------|
| 14 分档 | 13–15 分 | 译文精准还原原文含义，行文流畅、结构规整、用词贴切，基本无错，仅有个别小疏漏 |
| 11 分档 | 10–12 分 | 译文大体还原原意，结构较清晰、语句通顺，附带少量语言错误 |
| 8 分档 | 7–9 分 | 勉强译出原文大意，译文勉强连贯，语病繁多且包含部分严重错误 |
| 5 分档 | 4–6 分 | 仅能翻译原文少量内容，译文连贯性差，充斥大量严重语言错误 |
| 2 分档 | 1–3 分 | 只零散译出个别词语/短句，整体无法表达原文意思 |

### CET 分数换算

翻译满分 106.5 分（710 分的 15%）。近似换算同上表写作。

### 诊断维度（辅助分析，非官方评分）

| 诊断维度 | 说明 |
|---------|------|
| 信息完整度 | 原文关键信息是否全部译出 |
| 语义准确性 | 译文是否准确传达原文意思 |
| 语法正确性 | 译文语法是否规范 |
| 表达自然度 | 英语表达是否自然，有无中式英语 |
| 术语与文化表达 | 专有名词、文化概念的处理 |
| 语序与信息组织 | 是否按英语习惯重组信息 |

---

## 补充说明

- 四六级写作、翻译考核难度、考查要求不同，同一分数档次，四级与六级的评分样卷标准有级别区分。
- 反馈时，先给出官方档次判定，再用诊断维度拆解具体问题来源。

## Reading Scoring Rubric

阅读为客观题，按正确题数计分。以下为诊断维度，用于错因分析。

### 诊断维度

| 维度 | 说明 |
|------|------|
| 正确率 | 答题正确数 / 总题数 |
| 定位能力 | 能否准确定位答案在原文中的位置 |
| 推理准确性 | 推断题、主旨题、态度题的表现 |
| 干扰项识别 | 能否识别并排除干扰项 |
| 词汇理解 | 语境中的词汇理解能力 |

### 错因分类

| 错误类型 | 说明 |
|---------|------|
| 定位错误 | 找到原文相关位置但选错 |
| 推理过度 | 推断超出原文支持范围 |
| 概念偷换 | 被同义改写但改变范围的选项误导 |
| 因果倒置 | 混淆原因和结果 |
| 态度反转 | 将作者立场理解反 |
| 范围失当 | 选的范围太宽或太窄 |
| 关键词误导 | 被选项中的原文词汇误导 |
| 遗漏细节 | 忽略限定条件（时间、条件、程度） |
| 让步误判 | 将让步内容当作作者主论点 |
| 例子误判 | 将例子当作结论 |

## Listening Scoring Rubric

听力为客观题，按正确题数计分。以下为诊断维度，用于错因分析。

### 诊断维度

| 维度 | 说明 |
|------|------|
| 正确率 | 答题正确数 / 总题数 |
| 关键词捕捉 | 能否捕捉听力中的关键信息词 |
| 推理理解 | 对隐含信息、态度、意图的理解 |
| 干扰排除 | 能否排除干扰信息（近音词、跨段信息等） |
| 信息记忆 | 能否记住并整合分散的信息 |

### 错因分类

| 错误类型 | 说明 |
|---------|------|
| 近音误导 | 被发音相似的词误导 |
| 跨段混淆 | 选了另一段对话中的信息 |
| 态度反转 | 对说话人态度判断相反 |
| 时间错位 | 混淆事件发生的时间顺序 |
| 推理过度 | 添加了原文未提及的假设 |
| 遗漏转折 | 未抓住 but/however/actually 后的重点 |
| 数字混淆 | 混淆数字、时间、价格等信息 |
